# ProjectPacket

ProjectPacket helps freelancers and small creative agencies collect client project assets without chasing through email.

It uses Next.js, TypeScript, Tailwind CSS, lucide-react, Supabase Auth, Supabase Postgres, and Supabase Storage.

## Run

```bash
npm install
npm run dev
```

Open the local URL shown by Next.js.

## What To Test

1. Sign up.
2. Create a project.
3. Copy the client upload link.
4. Submit a file, text field, link, and approval from the client portal.
5. Return to the project detail page.
6. Open or download the uploaded file.
7. Approve an item.
8. Request changes on an item.
9. Restore sample data from settings.
